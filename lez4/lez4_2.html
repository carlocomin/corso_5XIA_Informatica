Chiusura di un insieme di attributi: X⁺

Definizione

Dati:

  • uno schema R
  • un insieme di DF F su R
  • un insieme di attributi X ⊆ R

la chiusura di X rispetto a F (X⁺) è:

tutti gli attributi determinabili a partire da X usando le DF in F.

Intuizione: X⁺ è “tutto ciò che posso sapere con certezza” se mi danno solo i valori di X e accetto le regole F come vere.

Procedura operativa (algoritmo)

  1. Inizia con X⁺ = X.
  2. Se esiste una DF Y → Z con Y ⊆ X⁺, allora aggiungi Z a X⁺.
  3. Ripeti finché non puoi più aggiungere attributi.
Input: X, F
Xplus := X
repeat
  changed := false
  for each (Y -> Z) in F do
    if Y ⊆ Xplus and Z ⊄ Xplus then
      Xplus := Xplus ∪ Z
      changed := true
until changed = false
Output: Xplus

Questo algoritmo è “a espansione”: parti da ciò che sai (X) e applichi tutte le DF che diventano attivabili man mano che la conoscenza cresce.

È il motore di quasi tutti i controlli: implicazione di DF, superchiavi, minimalità.

Esempio guidato

Schema R(A,B,C,D) e:

F = { A → B,  B → C,  CD → A }

Calcola A⁺ e (CD)⁺.

Calcolo di A⁺
  • Start: A⁺ = {A}
  • A → B ⇒ {A,B}
  • B → C ⇒ {A,B,C}
  • CD → A non applicabile (manca D)

A⁺ = {A, B, C}

Calcolo di (CD)⁺
  • Start: (CD)⁺ = {C,D}
  • CD → A ⇒ {A,C,D}
  • A → B ⇒ {A,B,C,D}

(CD)⁺ = {A, B, C, D} = R

Quindi CD è una superchiave di R.

Uso tipico di X⁺

  • Implicazione: X → Y è implicata da F se Y ⊆ X⁺.
  • Superchiave: X è superchiave se X⁺ contiene tutti gli attributi di R.
