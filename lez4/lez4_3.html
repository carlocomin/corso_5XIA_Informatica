X⁺ e F⁺: il ponte operativo

Due concetti diversi

NomeNotazioneDefinizione
Chiusura di attributi X⁺ (rispetto a F) tutti gli attributi determinabili a partire da X usando le DF di F
Chiusura di dipendenze F⁺ tutte le DF logicamente implicate da F (conseguenze)

In pratica F⁺ può essere enorme: non lo si enumera.

Intuizione: F è l’elenco “breve” di regole che ti danno; F⁺ è tutto ciò che puoi dedurre combinandole in tutti i modi possibili.

Caratterizzazione operativa (da ricordare)

Una DF X → Y è implicata da F se e solo se:

Y ⊆ X⁺

Quindi il calcolo di X⁺ è lo strumento pratico per decidere se una DF appartiene a F⁺.

Questo evita discussioni “a intuito”: invece di provare a indovinare, calcoli X⁺ e hai una risposta oggettiva.

Micro-esempio (implicazione)

Sia:

F = { A → B,  B → C }

Domanda: A → C è implicata?

Calcolo A⁺ rispetto a F:

  • Start: {A}
  • A→B ⇒ {A,B}
  • B→C ⇒ {A,B,C}

Poiché C ∈ A⁺, allora A → C è implicata.

Uso tipico (da oggi in poi)

  • Test “implicata?”: per verificare X → Y, calcola X⁺ e controlla Y ⊆ X⁺.
  • Superchiavi / chiavi: X è superchiave se X⁺ contiene tutti gli attributi di R.
  • Copertura minimale: i test “superfluo?” e “ridondante?” si fanno calcolando chiusure.
