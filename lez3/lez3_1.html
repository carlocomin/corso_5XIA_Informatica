Obiettivi e percorso — PHP + MySQL + HTML/CSS

Perché questo modulo

Integrare PHP, MySQL e HTML/CSS per costruire un flusso completo: richiesta del browser, elaborazione lato server, lettura/scrittura su database, risposta in pagina.

Obiettivo: collegare i concetti teorici del modello relazionale a un'implementazione web concreta.

Cosa porterai a casa

  • Backend: saper ricevere dati da form, validarli e persisterli in MySQL con prepared statements.
  • Frontend: impaginare tabelle e form con HTML/CSS mantenendo pulizia e usabilità.
  • Linguaggio: padroneggiare i costrutti fondamentali di PHP, con focus particolare su array e mappe associative.
  • Sicurezza di base: sanitizzazione input, escaping output, gestione credenziali.

Prerequisiti minimi

  • Concetti di HTTP e differenza tra client e server.
  • Ripasso rapido di HTML (form, tabelle, classi CSS) e del modello relazionale (tabelle, chiavi, vincoli).
  • Ambiente locale funzionante (PHP + MySQL) o stack equivalente (XAMPP/LAMP/WAMP/Containers).

Struttura della sequenza

  • Bootstrap rapido: setup, primo script PHP, flusso richiesta/risposta.
  • Basi del linguaggio: sintassi, tipi, controllo di flusso, funzioni.
  • Array e mappe: creazione, iterazione, ricerca, filtri e trasformazioni (pezzo centrale della lezione).
  • Interfacciamento al DB: connessione, query parametrizzate, gestione errori.
  • Integrazione front-end: render di risultati in HTML, styling essenziale con CSS.

Ogni step include esempi brevi + micro-esercizi per fissare i concetti.

Deliverable della lezione

  • Un mini-progetto che mostra dati da MySQL e inserisce record da form.
  • Snippet riutilizzabili per array/indexati e array associativi (mappe) con funzioni utili.
  • Checklist di sicurezza base per PHP + MySQL + HTML/CSS.
