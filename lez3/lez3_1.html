Obiettivi e percorso — PHP + MySQL + HTML/CSS

Perché questo modulo

Questo modulo introduce PHP come linguaggio lato server e lo collega subito a MySQL per costruire pagine che leggono input, eseguono query semplici e mostrano risultati in HTML.

Non e un modulo avanzato di backend: il focus e padroneggiare bene il flusso browser -> PHP -> MySQL -> HTML.

Cosa saprai fare alla fine

  • Scrivere script PHP semplici e leggibili integrati con HTML.
  • Leggere input da $_GET e $_POST, validarli e trattarli in modo essenziale ma corretto.
  • Aprire una connessione PDO verso MySQL.
  • Eseguire SELECT e INSERT di base con prepared statements.
  • Mostrare risultati del database in liste e tabelle HTML.

Cosa non faremo

  • Niente framework o architetture avanzate.
  • Niente ORM, routing complesso o pattern enterprise.
  • Niente ottimizzazioni da produzione: qui servono basi didattiche solide.

La priorita e capire il percorso del dato, non costruire un'applicazione complessa.

Prerequisiti minimi

  • Ripasso di HTML: form, tabelle e attributi principali.
  • Idea generale di client, server e richiesta HTTP.
  • Richiamo del modello relazionale: tabelle, chiavi e vincoli di base.

Sequenza del modulo

  1. Capire come viaggia una richiesta web e dove entra PHP.
  2. Scrivere PHP di base: output, variabili, tipi, array, funzioni.
  3. Leggere input da form e validarli.
  4. Collegarsi a MySQL con PDO.
  5. Eseguire query parametrizzate e mostrare il risultato in HTML.

Il filo conduttore sara sempre lo stesso: dati inseriti dall'utente, elaborati da PHP, salvati o letti da MySQL, ristampati in pagina.

Mini-progetto guida

  • Una pagina che legge righe da MySQL e le mostra in tabella.
  • Un form che raccoglie dati essenziali.
  • Un inserimento eseguito con prepared statement.
  • Una ristampa HTML dei dati aggiornati.

Deliverable attesi

  • Una base riutilizzabile per connessione PDO e query parametrizzate.
  • Snippet chiari per input, validazione minima e output HTML sicuro.
  • Un lessico tecnico coerente per distinguere linguaggio PHP, database e interfaccia web.
