Funzioni in PHP: idee fondamentali

Le funzioni servono a riusare codice e organizzarlo. In PHP non c'è un main: ogni file può contenere codice eseguibile e funzioni richiamabili.

Sintassi essenziale

Si definisce con la parola chiave function seguita dal nome e dalle parentesi. Il codice tra graffe viene eseguito quando la funzione è chiamata.

<?php
function saluta() {
    echo "Ciao a tutti!";
}

saluta(); // stampa "Ciao a tutti!"
?>

PHP legge l'intero file prima di eseguirlo: puoi chiamare una funzione anche se è dichiarata più in basso.

Parametri e valori di ritorno

  • I parametri si scrivono tra parentesi. Se manca un valore obbligatorio, PHP genera un errore.
  • Con return restituisci un risultato a chi chiama la funzione.
  • I tipi sono dinamici: puoi passare numeri o stringhe senza specificarli.
<?php
function somma($a, $b) {
    return $a + $b;
}

$risultato = somma(4, 6); // 10
?>

Valori di default

Puoi impostare un valore predefinito per rendere un parametro opzionale. I parametri con default vengono di solito alla fine della lista.

<?php
function salutaPersona($nome = "mondo") {
    return "Ciao $nome";
}

echo salutaPersona();        // Ciao mondo
echo salutaPersona("Luca"); // Ciao Luca
?>

Scope delle variabili

  • Le variabili create dentro una funzione esistono solo lì dentro.
  • Per leggere o modificare una variabile definita fuori, usa global oppure il super-array $GLOBALS.
<?php
$messaggio = "Ciao";

function stampaMessaggio() {
    global $messaggio;
    echo $messaggio;
}

stampaMessaggio(); // usa la variabile definita fuori
?>

Esempio pratico

Combinare parametri, ritorni e logica permette di costruire piccole utility riutilizzabili.

<?php
function calcolaMedia($lista) {
    $totale = array_sum($lista);
    $quantita = count($lista);
    return $quantita > 0 ? $totale / $quantita : 0;
}

$numeri = [6, 8, 10];
echo calcolaMedia($numeri); // 8
?>
